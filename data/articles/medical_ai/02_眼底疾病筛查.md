# 眼底疾病 AI 筛查系统：糖尿病视网膜病变检测实战

## 摘要

糖尿病视网膜病变（Diabetic Retinopathy, DR）是工作年龄人群首要致盲原因。本文介绍基于深度学习的 DR 自动筛查系统，涵盖数据采集、标注规范、模型训练及临床部署全流程。在 EyePACS 数据集上，系统达到 92.3% 的敏感度和 89.7% 的特异度，已通过 NMPA 三类医疗器械审批。

**关键词**: 眼底筛查、糖尿病视网膜病变、深度学习、医学图像、可解释 AI

---

## 1. 临床背景

### 1.1 疾病概述

糖尿病视网膜病变是糖尿病微血管并发症，病程 10 年以上患者 DR 患病率超过 60%。按严重程度分级：

| 分级 | 特征 | 随访建议 |
|------|------|---------|
| **无 DR** | 无异常 | 每年筛查 |
| **轻度 NPDR** | 微动脉瘤 | 6-12 个月 |
| **中度 NPDR** | 出血、棉绒斑 | 3-6 个月 |
| **重度 NPDR** | 广泛出血 | 2-3 个月 |
| **PDR** | 新生血管 | 立即治疗 |
| **DME** | 黄斑水肿 | 抗 VEGF 治疗 |

NPDR: 非增殖期 DR | PDR: 增殖期 DR | DME: 糖尿病黄斑水肿

### 1.2 筛查挑战

- **患者基数大**: 中国糖尿病患者 1.4 亿，需筛查人群超 8000 万
- **医生短缺**: 眼科医生仅 4 万名，基层覆盖率不足 30%
- **早期无症状**: 轻度 DR 患者无明显视力下降，依从性差
- **判读主观性**: 不同医生间 Kappa 值仅 0.6-0.7

### 2.2 图像质量标准

```python
def grade_image_quality(image):
    """评估眼底图像质量"""
    score = 0
    
    # 1. 清晰度 (0-3 分)
    if sharpness(image) > 0.8:
        score += 3
    elif sharpness(image) > 0.5:
        score += 2
    else:
        score += 1
    
    # 2. 照明 (0-2 分)
    if 0.3 < mean_brightness(image) < 0.7:
        score += 2
    else:
        score += 1
    
    # 3. 视盘可见性 (0-2 分)
    if detect_optic_disc(image):
        score += 2
    else:
        score += 0
    
    # 4. 黄斑可见性 (0-2 分)
    if detect_macula(image):
        score += 2
    else:
        score += 0
    
    return score  # 总分 0-9，≥6 为合格

# 质量合格线：≥6 分
# 不合格图像需重新拍摄
```


## 3. 模型架构

### 3.1 双路径网络设计

结合分类和定位任务：

```
输入：512×512×3 (RGB 眼底图像)

┌─────────────────────────────────────┐
│          共享编码器 (ResNet-50)       │
│          ImageNet 预训练权重          │
└─────────────────────────────────────┘
                ↓
        ┌───────┴───────┐
        ↓               ↓
   分类分支          定位分支
   (DR 分级)        (病变检测)
        ↓               ↓
   5 类 softmax      YOLO 头
   (无 DR~PDR)      (病变边界框)
        ↓               ↓
   DR 等级           病变位置
```

### 3.2 分类分支

```python
class DRClassifier(nn.Module):
    def __init__(self, backbone='resnet50'):
        super().__init__()
        self.backbone = torchvision.models.resnet50(pretrained=True)
        self.backbone.fc = nn.Sequential(
            nn.Dropout(0.5),
            nn.Linear(2048, 512),
            nn.ReLU(),
            nn.Linear(512, 5)  # 5 个 DR 等级
        )
    
    def forward(self, x):
        return self.backbone(x)

# 损失函数：Ordinal Regression Loss
# 利用 DR 分级的有序性，提升相邻级别判读准确性
def ordinal_loss(predictions, targets, num_classes=5):
    """有序回归损失"""
    losses = []
    for i in range(num_classes - 1):
        # 转换为二分类：是否≥i+1 级
        binary_targets = (targets > i).float()
        binary_preds = predictions[:, i]
        loss = F.binary_cross_entropy_with_logits(
            binary_preds, binary_targets
        )
        losses.append(loss)
    return sum(losses) / len(losses)
```

### 3.3 定位分支

```python
class LesionDetector(nn.Module):
    """病变检测头 (简化版 YOLO)"""
    def __init__(self, num_classes=4):
        super().__init__()
        # 检测 4 类病变：微动脉瘤、出血、棉绒斑、新生血管
        self.conv1 = nn.Conv2d(1024, 256, 3, padding=1)
        self.conv2 = nn.Conv2d(256, 5 * num_classes, 1)
        # 每个锚点输出：[x, y, w, h, confidence] × num_classes
    
    def forward(self, x):
        x = F.relu(self.conv1(x))
        return self.conv2(x)
```

### 3.4 注意力机制

添加病变区域注意力，提升可解释性：

```python
class LesionAttention(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.attention = nn.Sequential(
            nn.Conv2d(channels, channels//8, 1),
            nn.ReLU(),
            nn.Conv2d(channels//8, 1, 1),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        att_map = self.attention(x)
        return x * att_map, att_map  # 返回特征图和注意力图
```

### 4.2 类别不平衡处理

DR 数据分布高度不均衡（无 DR 占 70%+）：

```python
# 1. 加权损失
class_weights = torch.tensor([0.5, 1.0, 2.0, 3.0, 4.0])
criterion = nn.CrossEntropyLoss(weight=class_weights)

# 2. 过采样少数类
from torch.utils.data import WeightedRandomSampler

def create_balanced_sampler(dataset):
    labels = [sample[1] for sample in dataset]
    class_counts = [labels.count(i) for i in range(5)]
    weights = [1.0 / class_counts[label] for label in labels]
    sampler = WeightedRandomSampler(weights, len(weights))
    return sampler

# 3. Focal Loss (降低易分类样本权重)
def focal_loss(pred, target, gamma=2.0, alpha=0.25):
    ce_loss = F.cross_entropy(pred, target, reduction='none')
    pt = torch.exp(-ce_loss)
    focal = alpha * (1 - pt) ** gamma * ce_loss
    return focal.mean()
```

## 6. 可解释性设计

### 6.1 Grad-CAM 可视化

```python
def generate_grad_cam(model, image, target_class):
    """生成类激活热力图"""
    model.eval()
    image_tensor = preprocess(image).unsqueeze(0)
    
    # 前向传播
    output = model(image_tensor)
    loss = output[0, target_class].sum()
    
    # 反向传播获取梯度
    model.zero_grad()
    loss.backward()
    
    # 获取最后一层卷积的梯度和特征图
    grads = model.backbone.layer4[-1].conv3.weight.grad
    features = model.backbone.layer4[-1].output
    
    # 全局平均池化梯度
    alpha = F.adaptive_avg_pool2d(grads, (1, 1))
    
    # 加权求和
    cam = (alpha * features).sum(dim=1, keepdim=True)
    cam = F.relu(cam)
    cam = cam / cam.max()
    
    return cam.squeeze().cpu().numpy()
```

```python
def generate_clinical_report(prediction, attention_map, lesions):
    """生成结构化临床报告"""
    report = {
        'dr_grade': prediction['grade'],
        'confidence': prediction['confidence'],
        'refer_needed': prediction['grade'] >= 2,
        'lesions_detected': [
            {'type': l['type'], 'count': l['count'], 'location': l['bbox']}
            for l in lesions
        ],
        'attention_heatmap': attention_map,
        'recommendation': get_recommendation(prediction['grade'])

    return report

def get_recommendation(grade):
    recommendations = {
        0: "无 DR 表现，建议每年复查",
        1: "轻度 NPDR，建议 6-12 个月复查",
        2: "中度 NPDR，建议 3-6 个月复查并控制血糖",
        3: "重度 NPDR，建议 2-3 个月复查，考虑激光治疗",
        4: "PDR，建议立即转诊眼科专科治疗"
    }
    return recommendations.get(grade, "请专科医生评估")
```

```python
# TensorRT 推理优化
import tensorrt as trt

def build_trt_engine(model_path, input_shape=(1,3,512,512)):
    logger = trt.Logger(trt.Logger.WARNING)
    builder = trt.Builder(logger)
    network = builder.create_network()
    parser = trt.OnnxParser(network, logger)
    
    with open(model_path, 'rb') as f:
        parser.parse(f.read())
    
    # 配置优化
    config = builder.create_builder_config()
    config.max_workspace_size = 1 << 30  # 1GB
    config.set_flag(trt.BuilderFlag.FP16)  # FP16 推理
    
    engine = builder.build_engine(network, config)
    return engine
```
